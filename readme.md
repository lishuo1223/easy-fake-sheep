bv 显示区域中的 block ->路径，大小，是否被隐藏

1.将图片已经动态的渲染在页面 2.随机摆放图片，位置 ->坐标(x,y) 3.判断是否被遮挡：0 表示当前元素，1 表示目标元素 4.实现点击功能

- 获取到收集盒存放第一张图片的位置
- 判断收集盒为空直接放
- 不为空，是否有相同的元素存在
  - 如果没有，直接放入
  - 如果有，将所有元素向后移动一个位置，并将当前点击的元素插入到当前位置
    </br> 5.实现消除的功能
    _ 有 3 个相同的清楚掉
    _ 清楚掉之后，将后面的元素全部移上来
    _ 判断收集盒的元素达到 7 个，输了
    _ 收集盒的元素和 allblock 元素数量为 0，赢了
